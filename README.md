# 分结构眼球 3D 教学模型 3.1.2

3.1.2 是基于 3.1.1 的课堂交互修正版，保留 3.0 简化写实眼球 GLB，不改动解剖网格。

本版新增和修正：

- 明确标注电脑与触控设备的旋转、平移、缩放手势。
- 课堂预设改为相对模型的动态镜头，整体剖面和成像模式不再使用错误的远距离绝对坐标。
- 压缩课堂预设、结构分组、眼球状态和眼镜矫正区域。
- 新增独立视网膜成像观察窗，可显示上传图片的倒像及清晰/模糊变化。
- 光路增加眼镜平面的折点；近视配凹透镜、远视配凸透镜时，焦点可回到视网膜。
- 修正网址缺少成像参数时被误解析为 0 的默认值错误。

主要文件：

- `index.html`：在线查看器。
- `structured_eye_model.glb`：眼球结构模型。
- `eye-structures.json`：结构语义与教学说明。
- `vendor/`：Three.js 离线依赖。
- `qa-*.png`：桌面、平板、手机、上传图片及近视矫正截图。

验证基线：16 类结构、1973 个可拾取对象；桌面、平板和手机布局通过；HTML、JSON、GLB 均可经 HTTP 正常加载。
